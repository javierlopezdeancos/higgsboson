`hggs-tooltip`

  /*
  ... put here variables ...
  */
}

## HTML Structure

### Default

#### Basic HTML structure

```html
<span class="hggs-tooltip" data-tooltip="I am the text for the tooltip">
  Hover me
</span>
```
