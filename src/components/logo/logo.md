# Logo

## Root component class name

`hggs-logo`

## Theme selector

```css
.hggs-logo {
  /*
  ... put here variables ...
  */
}
```

## Component variables

```
--logo-height
--logo-height-small
--logo-url
--logo-url-dark
--logo-url-light
--logo-width
--logo-width-small
```

## HTML Structure

### Default

#### Basic HTML structure

```html
<div class="hggs-logo"></div>
```

#### Basic HTML structure with `--black` modifier

```html
<div class="hggs-logo hggs-logo--black"></div>
```

#### Basic HTML structure with `--white` modifier

```html
<div class="hggs-logo hggs-logo--black"></div>
```
